N8N Web Chatbot Integration Guide 🤖

Esta integración de chatbot te permite añadir fácilmente una interfaz de chat personalizable a tu sitio web, impulsada por flujos de trabajo de n8n.

¡Prueba el chatbot en vivo!

Haz clic en el icono de chat en la esquina inferior derecha para ver el chatbot en acción.

Características

Interfaz de chat en tiempo real

Comunicación fluida y en tiempo real con tus usuarios.

Modos ventana y pantalla completa

Flexibilidad para mostrar el chat como prefieras.

Apariencia personalizable

Adapta el diseño a la estética de tu sitio web.

Fácil integración con n8n

Conecta con tus flujos de trabajo existentes.

Compatibilidad con múltiples navegadores

Funciona en todos los navegadores modernos.

Requisitos previos

  • Una instancia activa de n8n con un webhook configurado
  • Un servidor web para alojar tu sitio
  • Conocimientos básicos de HTML

Inicio rápido

Añade este código a la sección <head> de tu HTML:

<link href="https://raw.githubusercontent.com/davidrumbaut620/N8N-WEB-CHATBOT-/refs/heads/main/chatbot.css" rel="stylesheet" />
<script type="module">
    import { createChat } from 'https://raw.githubusercontent.com/davidrumbaut620/N8N-WEB-CHATBOT-/refs/heads/main/chatbot.js';

    createChat({
        webhookUrl: 'TU_URL_DE_WEBHOOK_AQUÍ',
        showWindowCloseButton: true
    });
</script>

Opciones de instalación

Opción 1: Integración directa

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Sitio con Chatbot</title>
    <link href="https://raw.githubusercontent.com/davidrumbaut620/N8N-WEB-CHATBOT-/refs/heads/main/chatbot.css" rel="stylesheet" />
    <script type="module">
        import { createChat } from 'https://raw.githubusercontent.com/davidrumbaut620/N8N-WEB-CHATBOT-/refs/heads/main/chatbot.js';

        createChat({
            webhookUrl: 'https://n8n-production-b68b.up.railway.app/webhook/tu-id-único/chat',
            showWindowCloseButton: true
        });
    </script>
</head>
<body>
    <!-- Contenido de tu sitio web aquí -->
</body>
</html>

Configuración

Opciones de configuración básicas

Opción Descripción Valor predeterminado
webhookUrl Tu URL de webhook de n8n Requerido
showWindowCloseButton Mostrar/ocultar botón de cerrar true
mode Modo de visualización ("window" o "fullscreen") "window"
title Título de la ventana del chatbot "Chat"

Configuración avanzada

createChat({
    webhookUrl: 'TU_URL_DE_WEBHOOK_AQUÍ',
    showWindowCloseButton: true,
    mode: 'window',
    title: 'Atención al cliente',
    defaultLanguage: 'es',
    showWelcomeScreen: true,
    initialMessages: [
        "¡Hola! ¿En qué puedo ayudarte hoy?"
    ],
    i18n: {
        es: {
            title: "Chat de soporte",
            subtitle: "Estamos aquí para ayudarte 24/7",
            inputPlaceholder: "Escribe tu mensaje..."
        }
    }
});

Solución de problemas

Problemas comunes

Webhook no funciona

  • Verifica que la URL del webhook sea correcta
  • Comprueba si la instancia de n8n está en ejecución
  • Revisa la consola del navegador para ver errores

Problemas de estilo

  • Asegúrate de que el archivo CSS se cargue correctamente
  • Comprueba si hay conflictos CSS con tu sitio web

Problemas de integración

  • Verifica que todas las etiquetas de script estén en la sección <head>
  • Comprueba si hay errores en la consola de JavaScript
  • Asegúrate de que la carga del módulo sea correcta
